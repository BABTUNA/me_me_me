# me_me_me

Personal portfolio + blog. Built with Next.js 15 (App Router), Tailwind CSS, and MDX.

## Tabs

- **Home** — landing
- **Work** — projects
- **Blog** — writing
- **About** — bio + contact

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Stack

- Next.js 15, App Router, React 19
- Tailwind CSS v4
- MDX for blog posts
- TypeScript

## Deploy

Statically exportable. Hosts well on Vercel, Netlify, or Cloudflare Pages.
